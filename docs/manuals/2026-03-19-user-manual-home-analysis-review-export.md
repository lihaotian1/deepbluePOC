# API610 智能分析系统用户使用手册（全文智能比对、审核与导出流程）

## 一、进入系统并准备样例文件

1. 打开浏览器，访问系统首页地址。
2. 进入系统后，确认左侧导航高亮页签为`主页`。
3. 建议本次操作使用系统样例文件，以便观察全文比对、结果筛选、审核和导出流程。
4. 在本机找到样例文件`data/input/4503-GG-SS-018T01_A_Supply spec API610 Medium duty Pumps.pdf`。
5. 回到系统首页，定位到页面上方的`上传文件`区域。
6. 确认该区域内可以看到`选择文件`按钮和`确定`按钮。

## 二、上传文件并完成全文抽取

7. 点击`选择文件`。
8. 在系统文件选择窗口中选中样例 PDF 文件。
9. 点击系统文件选择窗口中的`打开`或等效确认按钮。
10. 回到页面后，确认`选择文件`按钮右侧文字已经变成`已选择: 4503-GG-SS-018T01_A_Supply spec API610 Medium duty Pumps.pdf`。
11. 点击`确定`。
12. 观察页面状态文本变为`正在抽取整篇文档...`或类似提示。
13. 等待文件上传与文本抽取完成。
14. 观察状态文本变为`文档上传完成，可开始全文智能比对`或同义提示。
15. 观察日志区出现`上传完成: 4503-GG-SS-018T01_A_Supply spec API610 Medium duty Pumps.pdf`。
16. 此时页面不会出现章节卡片，因为系统已改为整篇文档比对模式。

## 三、启动全文智能分析

17. 定位到`全文智能比对`区域。
18. 确认该区域中可以看到`开始智能分析`、`导出 Excel`和`提交审核`三个按钮。
19. 点击`开始智能分析`。
20. 观察状态文本变为`正在调用大模型进行全文比对...`或类似提示。
21. 观察日志区持续新增处理记录。
22. 日志中通常会出现以下类型的信息：
23. `已提交全文比对: xxx.pdf`
24. `识别到 P 类条目: ...`
25. `识别到 A 类条目: ...`
26. `识别到 B 类条目: ...`
27. `识别到 C 类条目: ...`
28. 等待系统完成全文智能分析。
29. 观察状态文本变为`智能分析完成，共命中 X 条结果`，或在无结果时显示`未识别到可提示条目`。

## 四、查看全文预览与结果表格

30. 定位到`分类筛选`区域。
31. 如需查看上传文件的完整内容，点击`查看全文`。
32. 观察页面展开`询价文件全文`区域。
33. 确认该区域展示的是完整抽取后的原文内容。
34. 点击`收起全文`，可恢复为仅看结果表格。
35. 定位到`命中结果`表格。
36. 确认表格列为：
37. `章节标题`
38. `询价文件的原文段落或句子`
39. `知识库标准化配套条目的原文`
40. `大模型总结的差异结论`
41. `分类`

## 五、按 P/A/B/C 进行筛选

42. 在`分类筛选`区域观察筛选按钮。
43. 确认系统仅显示以下筛选项：
44. `ALL`
45. `P`
46. `A`
47. `B`
48. `C`
49. 依次点击`P`、`A`、`B`、`C`，观察结果表格随筛选条件变化。
50. 点击`ALL`，恢复查看全部命中结果。
51. 注意：当前系统不显示未命中内容，因此也不再提供`其他 / OTHER`筛选项。

## 六、分页查看结果

52. 在结果表格下方查看分页条。
53. 确认分页条显示当前页码、总页数和总结果条数。
54. 点击`下一页`，观察系统切换到下一页结果。
55. 点击`上一页`，返回上一页。
56. 如当前命中条目较少，分页可能仅显示 1 页，这是正常现象。

## 七、打开结果抽屉并查看审核详情

57. 在结果表格中任选一条命中结果。
58. 点击该行。
59. 观察页面右侧弹出`审核详情`抽屉。
60. 在抽屉中确认可以看到以下信息：
61. 当前命中结果所属的章节标题。
62. `询价文件原文`。
63. `标准化配套条目`。
64. `差异结论`。
65. 当前结果的 `P/A/B/C` 分类标签。
66. `审核意见`输入框。

## 八、使用句段翻译功能

67. 在结果抽屉中定位到`询价文件原文`区域。
68. 点击`翻译句段`按钮。
69. 等待系统生成中文译文。
70. 观察原文区域切换为中文翻译结果。
71. 再次点击按钮，此时按钮会显示为`显示原文`或类似文案。
72. 观察内容切回原始语言原文。
73. 如翻译失败，抽屉中会显示`翻译失败，请重试`。

## 九、填写审核意见并标记已审

74. 在结果抽屉中的`审核意见`输入框中填写人工审核意见。
75. 例如：说明该结果是否需要向甲方澄清、由谁跟进、需补充哪些商务或技术说明。
76. 输入内容后，当前结果会自动回到`未审`状态。
77. 点击抽屉底部的`标记已审`。
78. 观察该条结果状态切换为`已审`。
79. 关闭抽屉后，在结果表格对应行中确认状态标签已更新。

## 十、删除单条结果

80. 如需移除某条模型结果，在其审核抽屉中点击`删除结果`。
81. 观察该条结果从表格中消失。
82. 注意：当前系统已移除“人工新增命中条目”功能，因此删除后不能直接在页面中补录新条目。

## 十一、提交审核

83. 回到页面上方的`全文智能比对`区域。
84. 点击`提交审核`。
85. 观察状态文本变为`正在提交审核...`或类似提示。
86. 等待系统提交完成。
87. 观察状态文本变为`文档已提交审核`。
88. 观察按钮区附近出现`已提交`标记。
89. 观察日志区出现`文档已提交审核`。

## 十二、导出 Excel 并检查结果

90. 保持当前页面停留在首页。
91. 点击`导出 Excel`。
92. 等待浏览器开始下载文件。
93. 下载完成后，在浏览器下载栏或系统下载目录中找到导出的 Excel 文件。
94. 打开导出的 Excel 文件。
95. 确认导出表头为：
96. `序号`
97. `章节标题`
98. `询价文件原文段落或句子`
99. `知识库标准化配套条目的原文`
100. `大模型总结的差异结论`
101. `分类`
102. `审核意见`
103. `审核状态`
104. 找到你刚刚审核过的对应结果行。
105. 确认该行的`审核意见`列已写入人工填写内容。
106. 确认该行的`审核状态`列显示为`已审`。
107. 注意：导出结果只包含命中条目，不会补`OTHER`或未命中内容。

## 十三、维护标准化配套知识库

108. 点击左侧导航中的`知识库`。
109. 进入后确认页面标题为`标准化配套知识库`。
110. 当前页面仅维护`标准化配套知识库.json`，不再提供多文件切换、新建或删除功能。
111. 在搜索框中输入关键词，可按分类名、条目内容或类型值检索。
112. 如需新增分类，点击页面上方的`新增分类`。
113. 如需在某分类下新增条目，点击该分类卡片右上角的`新增条目`。
114. 如需修改条目，直接编辑条目原文和对应的`P / A / B / C`值。
115. 如需删除条目，点击条目行右侧的`删除`。
116. 编辑完成后点击`保存知识库`。
117. 观察页面提示`保存成功`。

## 十四、使用限制说明

118. 当前系统仅支持`标准化配套知识库`参与比对。
119. 当前系统采用整篇文档一次性送模型的方式，不再做文档切分。
120. 如果上传文档超过当前模型上下文上限，系统会直接报错，不会自动拆分或摘要。
121. 当前系统不支持在页面中对原始全文进行编辑。
122. 当前系统不支持人工新增命中条目。
123. 当前系统不显示未命中内容，因此没有`OTHER`分类和对应导出行。
